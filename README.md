## **GAN (Generative Adversarial Network)**
A **generative adversarial network (GAN)** is a machine learning model that uses two neural networks to generate new data. GANs are a prominent framework for generative artificial intelligence (AI).

### **How GANs work?**
* **Generator:** A neural network that creates fake data <br>
* **Discriminator:** A neural network that identifies the fake data created by the generator <br>
* **Zero-sum game:** The generator and discriminator compete against each other in a zero-sum game, where one's gain is the other's loss <br>

### **What GANs can do?**
* **Create new images:** GANs can generate new images from an existing image database <br>
* **Create new music:** GANs can generate new music from a database of songs <br>
* **Create 3D models:** GANs can generate 3D models of objects and scenes from 2D images <br>
* **Upscale images:** GANs can upscale low-resolution images to high-resolution images <br>

### **When GANs are used ?**
* Quality data generation <br>
* Unsupervised learning <br>
* Predictive applications <br>
* Virtual reality <br>
* Video games <br>
* Computer-aided design <br>

### **How to train GANs ?**
* Use a diverse set of training data <br>
* Monitor the training process <br>
* Ensure the generator and discriminator networks are well-balanced <br>
